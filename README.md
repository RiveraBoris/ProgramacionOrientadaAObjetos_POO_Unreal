# ProgramacionOrientadaAObjetos_POO_Unreal
# Ingeniería de Sistemas – Aprendizaje en Programación y Desarrollo de Software

## 📚 Descripción

Este repositorio forma parte del proceso de formación académica de un **estudiante de Ingeniería de Sistemas**, enfocado en el aprendizaje y la práctica de **programación**, **estructuras de software** y **desarrollo de aplicaciones interactivas**.

El objetivo principal es documentar proyectos, ejercicios y experimentos realizados durante el proceso de aprendizaje, especialmente en áreas relacionadas con:

* Programación en **C++**
* **Programación Orientada a Objetos (POO)**
* Lógica y estructuras de programación
* Desarrollo de sistemas
* Introducción al desarrollo de videojuegos con **Unreal Engine**

Este espacio funciona como un **portafolio de aprendizaje**, donde se registran avances, prácticas y proyectos desarrollados durante la formación académica.

---

## 🎯 Objetivos

* Desarrollar habilidades sólidas en **programación**.
* Comprender y aplicar los principios de **Programación Orientada a Objetos**.
* Aprender a diseñar y estructurar **software modular y escalable**.
* Explorar herramientas utilizadas en la industria del software.
* Introducir conceptos de **desarrollo de videojuegos y simulaciones**.
